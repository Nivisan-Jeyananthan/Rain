# Serialisierung / Deserialisierung Projekt
Dieses Projekt zeigt, wie Serialisierung und Deserialisierung erzielt werden kann.   
Es ist eine Variation davon.

Zurzeit werden keine verschachtelten Objekte unterstüzt.

# Übersicht
Alle Objekte, seien es Primitive oder echte Objekte werden auf Byte eben aufgeteilt.  
Es zerlegt die jeweiligen Daten so auf, dass Sie in einem gewöhnlichen Byte format über Netzwerk später  
oder auch auf einer Festplatte überträgen/abgespeichert werden können.
